# ANS-Rechorus
## 基于Rechorus框架的ANS算法复现
### Getting Started
  '''bash
  git clone https://github.com/LongMaxSilver6247/ANS-Rechorus.git
  cd ANS-Rechorus
  '''

在ANS-Rechorus目录下创建虚拟环境
  '''bash
  pip install -r requirements.txt
  cd src
  '''
运行模型
    '''bash
    python main.py --model_name ANS
    '''
